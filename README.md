# Hydrogen

The monolithic repository for Planet Pope and all the services that supports the Content Management System and the Planet Pope mobile application.

## Features

### User Management:

- User details
- Authorization
- Authentication
- Roles
- Permissions

### Content Management:

- Vidoes
- Songs
- Images
- Documents (Journals)

### Project Management:

- Managing details

### System Settings:

- Settings Update system-wide

### Event Management:

- Registration
- Managing details

### Subscription Management:

- Black Boys Read (Free)
- Be a buddy & make a difference (Free)
- Self Love Project
- Music

### Analytics:

- Engagement
- _Estimator_

### Taxonomy Management:

- Tags

### Financial Reporting:

- Budgets
- Reports (Expenses, etc)
